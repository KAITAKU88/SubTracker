# SubTracker — Hướng dẫn nhanh

## Đăng bản APK mới (chỉ 3 bước)

1. Mở **GitHub** → repo **SubTracker** → tab **Actions**
2. Chọn **Publish APK** (bên trái) → nút **Run workflow** → **Run workflow**
3. Đợi khoảng 10–15 phút. Xong!

Không cần cài Java, Android SDK hay chạy lệnh gì trên máy.

## Link (không đổi)

- **Landing page:** https://kaitaku88.github.io/SubTracker/
- **Tải APK:** https://github.com/KAITAKU88/SubTracker/releases/latest/download/subtracker.apk

Mỗi lần bấm Publish APK, GitHub tự:
- Tăng số phiên bản
- Build file APK
- Đăng lên Releases (tên file luôn là `subtracker.apk`)

## Landing page tự cập nhật

Khi sửa file trong thư mục `docs/` và push lên GitHub → trang web tự deploy lại.
